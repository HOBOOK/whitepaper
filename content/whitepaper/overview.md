# 시스템 개요

플랫폼은 데이터 수집/처리, 시뮬레이션, 응용 서비스 레이어로 구성됩니다.

## 데이터 레이어
- 필드 게이트웨이(OPC‑UA, Modbus, MQTT)
- 이벤트 스트림/버스(Kafka/Redpanda 계열)
- 시계열/도메인 저장소

## 시뮬레이션 레이어
- 3D/물리 시뮬레이터(Isaac‑Sim/Lab)
- 시나리오/버전 관리

## 서비스 레이어
- API 게이트웨이/인증
- 모니터링/알림/리포팅
- 운영 자동화(에이전트)
